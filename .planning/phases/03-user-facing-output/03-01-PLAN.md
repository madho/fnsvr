  <action>
Create src/fnsvr/notifier.py with these functions:

1. `notify(title: str, message: str, subtitle: str = "", sound: str = "Pop") -> bool`:
   - Check `platform.system() == "Darwin"`. If not macOS, log debug message "Notifications not supported on {platform}" and return False (per NOTF-05).
   - Build osascript command: `osascript -e 'display notification "{message}" with title "{title}" subtitle "{subtitle}" sound name "{sound}"'`
   - Escape double quotes in title/message/subtitle by replacing `"` with `\\"`.
   - Truncate message to 200 chars if longer (notification display limit).
   - Run via `subprocess.run(["osascript", "-e", script], capture_output=True, timeout=5)`.
   - Catch all exceptions (subprocess.TimeoutExpired, OSError, etc.), log warning, return False. Notifications must NEVER block scanning.
   - Return True on success.

2. `notify_detections(detections: list[dict], config: dict) -> None`:
   - If `config.get("notifications", {}).get("enabled", True)` is False, return immediately (per NOTF-04).
   - If not on macOS (platform.system() != "Darwin"), log debug and return (per NOTF-05).
   - `detections` is a list of dicts with keys: category, priority, subject, account_name.
   - Get `batch_threshold` from `config.get("notifications", {}).get("batch_threshold", 5)`.
   - Get sound names from config: `critical_sound` (default "Submarine"), `normal_sound` (default "Pop").
   - If `len(detections) == 0`, return.
   - If `len(detections) > batch_threshold` (per NOTF-03): send ONE summary notification with title "fnsvr: {count} new detections", message listing count by priority (e.g., "2 critical, 3 high"), sound = critical_sound if any detection is critical, else normal_sound.
   - If `len(detections) <= batch_threshold`: send individual notifications. For each detection, title = "fnsvr: {category_label}", message = truncated subject, subtitle = account_name, sound = critical_sound if priority == "critical" else normal_sound (per NOTF-01, NOTF-02).

Use `logging.getLogger(__name__)` for all log messages. Import platform, subprocess, logging.

Module docstring: "macOS native notifications for fnsvr via osascript."
