Add a new function `get_emails_by_date_range` after `get_unreviewed`:

```python
def get_emails_by_date_range(
    conn: sqlite3.Connection,
    days: int = 7,
    unreviewed_only: bool = False,
) -> list[sqlite3.Row]:
    """Get emails from the last N days, ordered by priority then date."""
    query = "SELECT * FROM detected_emails WHERE created_at >= datetime('now', ?)"
    params: list = [f"-{days} days"]
    if unreviewed_only:
        query += " AND reviewed = 0"
    query += " ORDER BY priority ASC, date_received DESC"
    return conn.execute(query, params).fetchall()
```

Functions:

1. `generate_digest(emails: list[dict], title: str = "fnsvr Digest") -> str`:
   - Takes a list of email dicts (or sqlite3.Row objects with dict-like access).
   - Returns a markdown string.
   - Structure (per DIG-02, DIG-03, DIG-04):
     a. `# {title}` + `Generated: {datetime.now().strftime("%Y-%m-%d %H:%M")}` + blank line
     b. `## Summary` section: total count, then "By Priority:" bullet list counting critical/high, then "By Account:" bullet list counting per account_name.
     c. For each category in CATEGORY_ORDER that has emails: `## {CATEGORY_LABELS[cat]}` heading, then for each email in that category render: `### {subject}` with bullet points for From, Date, Account, Priority, Attachments (Yes/No based on has_attachments), and Snippet (first 200 chars) if present.
     d. `## Action Required` section: filter emails where `reviewed == 0` and `priority == "critical"`. List each as `- **{subject}** ({account_name}) -- {category}`. If none, write "No critical unreviewed items."
   - If `emails` is empty, return title + "No detections in this period."

2. `save_digest(content: str, config: dict, no_save: bool = False) -> Path | None`:
   - If `no_save` is True, return None (per DIG-07, caller prints to stdout).
   - Resolve digests path from `config["paths"]["digests"]` using `config_module.resolve_path`.
   - Create directory if needed.
   - Filename: `digest_{datetime.now().strftime("%Y-%m-%d")}.md`
   - Write content to file.
   - If `config.get("digest", {}).get("obsidian_copy", False)` is True (per DIG-06):
     - Resolve `config["digest"]["obsidian_path"]` via config_module.resolve_path
     - Create directory if needed
     - Copy file there using shutil.copy2
     - Log info message about Obsidian copy
   - Return the primary save path.
