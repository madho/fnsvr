Create src/fnsvr/scanner.py with these exact functions and behaviors:

**Module-level constant (per AUTH-02):**
```python
SCOPES = ["https://www.googleapis.com/auth/gmail.readonly"]
```
This is the ONLY scope. No write access under any circumstances.

**setup_oauth(account: dict, config_dir: Path) -> bool (AUTH-01, AUTH-03):**
- Resolve credentials_file path: `config_dir / account["credentials_file"]`
- Resolve token_file path: `config_dir / account["token_file"]`
- If credentials file does not exist, raise FileNotFoundError with message: `f"Credentials file not found: {creds_path}\nDownload OAuth client credentials from Google Cloud Console."`
- Call `InstalledAppFlow.from_client_secrets_file(str(creds_path), SCOPES)`
- Call `flow.run_local_server(port=0)` to open browser
- Create parent dirs for token: `token_path.parent.mkdir(parents=True, exist_ok=True)`
- Write token: `token_path.write_text(creds.to_json())`
- Set permissions: `os.chmod(str(token_path), 0o600)` (per AUTH-03)
- Return True on success

**get_gmail_service(account: dict, config_dir: Path) -> service | None (AUTH-04, AUTH-05):**
- Resolve token_path from `config_dir / account["token_file"]`
- If token_path does not exist, return None
- Load: `Credentials.from_authorized_user_file(str(token_path), SCOPES)`
- If creds exist and creds.expired and creds.refresh_token:
  - Try: `creds.refresh(Request())`
  - Rewrite token file with `creds.to_json()` and `os.chmod(str(token_path), 0o600)`
  - On exception (catch google.auth.exceptions.RefreshError specifically):
    - Log: `logger.error(f"Token refresh failed for {account['name']}. Re-run: fnsvr setup {account['name']}")`
    - Return None
- If not creds or not creds.valid, return None
- Return `build("gmail", "v1", credentials=creds)`

**get_header(headers: list[dict], name: str) -> str:**
- Iterate headers list (Gmail format: [{"name": ..., "value": ...}])
- Case-insensitive name comparison
- Return value or empty string if not found

**build_query(lookback_days: int) -> str (SCAN-01, SCAN-02, SCAN-03):**
- Calculate epoch: `int(time.time()) - (lookback_days * 86400)`
- Return `f"after:{epoch}"`
- Use epoch timestamps, NOT date strings (avoids timezone ambiguity per research)

**fetch_message_ids(service, query: str, max_results: int) -> list[str]:**
- Call `service.users().messages().list(userId="me", q=query, maxResults=min(max_results, 500))`
- Handle pagination via nextPageToken
- Truncate at max_results (hard cap)
- Return list of message ID strings
- If response has no "messages" key, return empty list

**scan_account(account: dict, config: dict, conn: sqlite3.Connection, patterns: list[CompiledCategory], lookback_days: int, config_dir: Path) -> tuple[int, int, int] (SCAN-06):**
- Call get_gmail_service; if None, raise RuntimeError with AUTH-05 message
- Insert scan_log via storage.insert_scan_log(conn, account["name"], account["email"], datetime.now(timezone.utc).isoformat())
- Build query with build_query(lookback_days)
- Fetch message IDs with fetch_message_ids(service, query, config["scan"]["max_results_per_scan"])
- For each message_id:
  - Fetch full message: `service.users().messages().get(userId="me", id=msg_id, format="full").execute()`
  - Extract subject via get_header(payload["headers"], "Subject")
  - Extract sender via get_header(payload["headers"], "From")
  - Extract date via get_header(payload["headers"], "Date")
  - Extract snippet from message.get("snippet", "")
  - Check has_attachments: any part in payload that has a "filename" with non-empty value (walk parts recursively)
  - Run detector.match_email(subject, sender, patterns)
  - If match: call storage.insert_email(conn, {...email dict...}) with all fields
  - Wrap individual message processing in try/except to skip broken messages
- Update scan_log via storage.update_scan_log with completed_at, emails_scanned, emails_detected, status
- If any errors occurred, set status="completed_with_errors" and errors=joined error strings
- Return (scanned_count, detected_count, 0) -- attachment downloads handled separately by downloader

**scan_all(config: dict, conn: sqlite3.Connection, lookback_days: int, config_dir: Path, account_filter: str | None = None) -> list[tuple] (SCAN-04, SCAN-05):**
- Call detector.compile_patterns(config["categories"]) once
- If account_filter is not None, filter config["accounts"] to matching name (per SCAN-04)
- For each account (per SCAN-05):
  - try: call scan_account, append (name, scanned, detected, 0, None)
  - except Exception as exc: append (name, 0, 0, 0, str(exc))
  - Log error but continue to next account
- Return results list
